### STEPS INSTALL AND RUN CODE
- Step 01: Clone repository:

```
git clone <https://github.com/TranLeKhanhDuyen/intern-agility.git>
```

- Step 02: Move to folder BookingCare:

```
cd BookingCare
```

- Step 03: Install dependencies:

```
npm install
```

- Step 04: Finally run with:

```
npm run dev
```
